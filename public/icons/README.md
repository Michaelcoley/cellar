# Icons

Drop the following PNGs in this folder before deploying:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512) — also referenced from the README
- `icon-512-maskable.png` (512×512, with safe-zone padding)

Generate from the SVG at `public/favicon.svg` or your own master artwork. Tools:

- https://realfavicongenerator.net
- https://maskable.app
